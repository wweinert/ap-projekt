# Mobile Goods Inspection App

## Overview
Mobile application for digital goods inspection, including documentation, image capture and automated PDF report generation.

## Features
- Digital inspection workflow
- Photo documentation
- PDF report generation
- Role-based user management

## Tech Stack
- React Native
- NestJS
- PostgreSQL
- TypeScript

## My Contribution

- Backend API development
- Data handling and reporting logic
